# Frontend React JS (Vite + Yarn)

## Description

Frontend du projet web de M1 1er semestre. En **React.js** avec **Vite** comme build tool et **Yarn** comme gestionnaire de packages.  

Au lancement du container Docker, le serveur de dev se lance de lui-meme, accessible sur le port `8081` (http://localhost:8081/)

Si le serveur ne se lance pas ou se ferme, vous pouvez le relancer avec la commande suivante : `yarn dev`.
